js_terminal
===========

A fake javascript terminal


Demo
====

http://oros42.github.io/js_terminal/


Key
===

```Tab``` : autocomplete  
```Up``` : back in history  
```Down``` : forward in history  
```Alt```+```Backspace``` : remove word  
```Ctrl```+```d``` : exit  


Commands
========

```help```   
```ls [dir]```   
```cd <dir>```   
```cat <file>```   
```pwd```   
```mkdir <dir>```   
```man```   
```clear```   
```exit```   
```echo "<text> > <file>"```
